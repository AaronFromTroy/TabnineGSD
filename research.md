# Research Workflow

This file defines how research is performed before implementation.

## Research Sources
- Workspace files
- Architecture.md
- Tasks.md
- External documentation (only when referenced by the user)

## Research Questions
- What is unknown?
- What libraries or APIs are involved?
- What risks exist?
- What dependencies must be resolved?

## Output Format
- Summary of findings
- Risks identified
- Recommendations
- Questions for the user
