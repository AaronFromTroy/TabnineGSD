# GSD Workflow

This file defines the full Guided System Development loop.

## Step 1 — Ask for the Goal
“What do you want to accomplish?”

## Step 2 — Create Phases
Break the goal into 3–7 phases.
Ask clarifying questions.

## Step 3 — Research
Perform a research pass.
Summarize findings.
Ask if the user wants to incorporate them.

## Step 4 — Expand Tasks
Break phases into tasks.
Ask clarifying questions.

## Step 5 — Execute
Generate code or artifacts.
Validate against the spec.

## Step 6 — Iterate
Ask:
“Revise, continue, or next phase?”
